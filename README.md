# UHC

Plugin for Minecraft 1.16.4 developed by NakeSeal

**Features:**

**Commands:**\
`prepare`Prepare the game zone.\
`team`Commands for team management.\
`tc`Team chat.\
`inv`Open a player inventory.\
`report`Report a player.
